# Vote System (website)
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```
## Serve
### Build as docker (docker-compose)
```bash
# deploy to docker (docker-compose)
$ npm install
$ docker-compose -p "web" up -d
```
### Access URL http://localhost/, http://127.0.0.1/ ☕️
